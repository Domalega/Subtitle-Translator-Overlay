# Task Template

Use this prompt for most tasks:

```text
Экономно.

Задача:
<описание>

Ограничения:
- читай минимально нужные файлы;
- не сканируй весь проект;
- не читай dist/node_modules;
- не запускай build/tests/git;
- не трогай несвязанный код;
- после завершения кратко опиши изменения.
```

For investigation first:

```text
Экономно. Найди минимальный набор файлов для этой задачи. Ничего не меняй. Не запускай build/tests/git. Кратко дай план.
```
